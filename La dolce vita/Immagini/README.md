Questo folder contiene le immagini e le risorse (PDF, SVG) usate dall'interfaccia.

Per completare il progetto copia i file dalla cartella del progetto originale:

Percorso sorgente (nel workspace attuale):
`/Users/mac/Desktop/La dolce vita/static/Immagini/`

File importanti da copiare qui:
- `hollowed-boxes.svg`
- `Screenshot 2025-11-18 alle 02.32.53.png` (o l'immagine che usi per la preview)
- eventuali PDF come `Registro Locale_V1 2025-2026.pdf`

Puoi copiare i file con un comando terminale (macOS):
```
mkdir -p "La dolce vita-vbnet/wwwroot/Immagini"
cp /Users/mac/Desktop/La\ dolce\ vita/static/Immagini/* "La dolce vita-vbnet/wwwroot/Immagini/"
```
